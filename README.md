# F-RevoCRM API Sample
## init
```
git clone git@github.com:Remicck/F-RevoCRM-API-Sample.git
cd F-RevoCRM-API-Sample

npm install
# edit your own F-RevoCRM URL, UserName and Accesskey
node index.js
```
